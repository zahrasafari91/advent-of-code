# advent-of-code
Calendar of Small Programming Puzzles

This repository contains my solutions to the [Advent of Code](https://adventofcode.com/) programming challenges.

The goal is to practice:

* Problem-solving
* Algorithms and data structures
* Clean Python programming
* Code optimization


## Running a Solution

Open the required day folder and run:

```bash
python solution.py
```

## Notes

Each solution folder contains:

* The Python solution
* A short problem summary
* The approach and complexity
* Instructions for running the code

Puzzle descriptions and inputs belong to Advent of Code.
